// npm

install → npm install packageName
uninstall → npm uninstall packageName

any particular version install or uninstall:
npm install packageName@version
npm uninstall packageName@version

ex: npm i accessibility@4.5.12

// dependencies:
all package called dependencies in project. 


ex: express dependent on many package. when we install express, all needed package installed at this time.
when we delete express, all dependent package is deleted



// devdependencies:

jesob packages kebol development er time a use kora hoy, 
app complete hoyer por ar kaje lage na, seisob package ke devdependencies bola hoy

ex: nodemon

app deploy er por devdependencies proyojon pore na.

so, those package is not needed after deploy. 
we will keep those packages at devdependencies

start, test, stop, restart = npm auto recognizes these name. 
they are shortcut of → npm run start/these
